🚀 БЫСТРЫЙ СТАРТ (прямо сейчас!)
═══════════════════════════════════════════════════════════════

Шаг 1: Установите зависимости
    $ pip install -r requirements.txt

Шаг 2: Создайте .env файл
    $ cp .env.example .env
    $ nano .env

    Добавьте:
    CLIENT_BOT_TOKEN=ваш_токен_от_BotFather
    ADMIN_BOT_TOKEN=ваш_токен_админ_бота
    ADMIN_PIN=1234

Шаг 3: Запустите бота
    $ python main.py --config configs/client_lite.json

🎉 Готово! Бот работает!
